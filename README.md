# 🎨 My Pixel Life

### 📝 Project Overview
"Jessica — a mischievous girl who is curious about everything. She has a hundred different interests."

This is an interactive pixel digital room designed to showcase Jessica's personality, hobbies, and life journey.

---

### ✨ Key Features
* **Interactive Environment:** Explore clickable pixel-art objects.
* **Character Narrative:** Discover the stories behind the items in the room.
